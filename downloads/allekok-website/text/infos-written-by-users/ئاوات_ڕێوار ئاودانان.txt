ڕێوار ئاودانان، خەڵک شار ئاودانان سەر وە پارێزگاێ ئیلام ڕۆژهەڵات کوردستانە. دەناو بنەماڵەێ ئەژ کوردەل پەهڵەیی ( کوردەلی ـ فەیلی) دە ساڵ ١٩٨٠ زایینی (١٣٥٩ هەتاوی) وەدی هاتێیە. دە زانکۆ کرماشان خوەنسێیە. دە ساڵ ٢٠٠٤ بەشدار جمشت ئاپۆیی بییە. چەن پرتووک شێعر و هەڵبەس ئەژ ئی کەسایەتیی کوردە وە زاراوەێ کوردیی خوارین (پەهڵەیی- کەڵهوڕی- لەکی) چاپ و بڵاو بینە.
ناو پرتووکەیل هەڵبەس ڕێوار ئاودانان:

١- چەمێ ئەژ چریکەێ بەرد (کوومە هەڵبەس ١ )

٢- گوڵدەم سۊر (کوومە هەڵبەس ۲ )

٣- دەمتەقە وە‌گەرد باێ (کوومە هەڵبەس ۳ )

٤- ئی زامە، ئاشارەو نێیەو (کوومە هەڵبەس ۴ )

٥- سەرەوژێری (کوومە هەڵبەس ۵ )

٦- بۊش کامە وەرزی؟ (کوومە هەڵبەس ۶ )

٧- پاش واران (کوومە هەڵبەس ٧ )

٨- وشەێل، ئەژ هەفت دەلیا گوارننەم
end
ڕێوار ئاودانان، خەڵکی شاری ئاودانان سەر بە پارێزگای ئیلام لە ڕۆژهەڵاتی کوردستانە. لە کوردانی پەهڵەیی (فەیلی)ە. ساڵی 1980 زایینی (١٣٥٩ هەتاوی) لە دایک بووە.
چەند پەرتووکی شێعر و هەڵبەستی هەیە ک بە زاراوەی کوردیی خوارین (پەهڵەیی- کەڵهوڕی- لەکی) نووسراون.
